# NLP for Indic Languages — Transformer Fine-Tuning

Independent research project fine-tuning Transformer models for Tamil and other Indic languages.

**Status: Active / In Progress** — started in 2026 jun

## Why this project

Tamil has 250M+ speakers, yet functional NLP tooling in the language is minimal compared to English or other major world languages. This gap affects everything from customer support automation to government service accessibility to voice tools for low-literacy users.

## What I'm doing

- Fine-tuning pretrained Transformer models (via HuggingFace) on Tamil-language text data.
- Built a preprocessing pipeline to automate repetitive annotation steps, reducing manual labelling time across the dataset.
- Targeting practical applications: Tamil-language customer support, accessible government documents, and voice tools for low-literacy users.

## Current stage

Dataset preparation and model fine-tuning in progress. Findings are being prepared for submission to a student NLP workshop.

## Technologies

Python, HuggingFace Transformers, PyTorch
